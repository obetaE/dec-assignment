Quick Review: What You Already Know

Before we dive into flexbox, let's review what you already know about the CSS display property:

display: block

Elements stack vertically, take full width

div, p, h1, section

display: inline

Elements flow horizontally, width based on content

span, a, strong, em

display: none

Completely hides element from page

visibility: hidden;

The Problem

Block and inline have limited layout options. What if we need more control?

Introducing display: flex

display: flex is a CSS property value that turns an element into a flex container. Once an element is a flex container, its direct children automatically become flex items that you can easily arrange, align, and distribute.

üì¶ Moving Boxes Analogy

Imagine you have three boxes on the floor:

  • Without flex: Boxes stack vertically (like display: block)
  • With flex: You get a smart shelf that can arrange boxes horizontally, vertically, centered, or spaced evenly
  • The magic: The shelf (flex container) controls how all boxes (flex items) are arranged

With display: flex, you're telling the browser: "Hey, I want all my children elements to be flexible and follow my layout rules!" This one line of code unlocks powerful layout capabilities that were previously difficult or impossible with traditional CSS.

How Flexbox Works

Two-Level System

Flexbox works on a two-level system. Understanding this is key to mastering flexbox:

1. Flex Container

The parent element with display: flex. It controls the layout of its children.

.container { display: flex; }

2. Flex Items

All direct children of the flex container. They become flexible and follow container rules.

.item { /* automatically flex items */ }

HTML Structure Example

<!-- The container with display: flex -->
<div class="container">
  <!-- These are the flex items -->
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

Remember: Only direct children become flex items. Grandchildren are not affected.
