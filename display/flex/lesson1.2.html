Understanding Flex Direction

flex-direction is a CSS property that controls the main axis of your flex container. It tells flex items which direction to flow: horizontally, vertically, forward, or backward.

üìö Book Reading Analogy

Think of flex-direction as choosing your reading direction:

  • row: Like reading English - left to right
  • row-reverse: Like reading Arabic - right to left
  • column: Like reading Japanese manga - top to bottom
  • column-reverse: Like scrolling up a chat - bottom to top

Remember: Flexbox works on two axes - the main axis (primary direction) and the cross axis (perpendicular direction). flex-direction determines which is which!

The Two Axes: Main vs Cross

Visual Guide

Understanding axes is crucial for mastering flexbox. Here's how they work:

Main Axis

The primary direction where flex items flow. Controlled by flex-direction.

Main Axis
‚Üí
flex-direction: row;

Controls item flow direction

Cross Axis

The perpendicular direction. Used for alignment across the main axis.

Cross Axis
‚Üì
align-items: center;

Controls vertical alignment

Key insight: When you change flex-direction, you're swapping which axis is main and which is cross! row makes horizontal the main axis, column makes vertical the main axis.

Practice Challenge: Mobile Layout

Your Mission: Create a responsive layout that changes direction based on screen size.

  1. Start with the horizontal layout (default row)
  2. Your goal: Switch to vertical layout (column) when needed
  3. This mimics how mobile sites stack elements vertically
  4. Test both layouts using the buttons below
  5. Extra credit: Try the reverse variations too!

/* Responsive layout with media queries */
.container {
  display: flex;
  flex-direction: row; /* Desktop: horizontal */
  gap: 20px;
}

/* Mobile layout */
@media (max-width: 768px) {
  .container {
    flex-direction: column; /* Mobile: vertical */
  }
  
  .item {
    width: 100%; /* Full width on mobile */
  }
}
