Introducing justify-content

justify-content is a CSS property that controls how flex items are distributed and aligned along the main axis of the flex container. It answers the question: "How should the available space be distributed around or between my flex items?"

üé™ Theater Seating Analogy

Imagine arranging chairs in a theater row:

  • flex-start: Push all chairs to the left side
  • center: Center all chairs in the row
  • flex-end: Push all chairs to the right side
  • space-between: Equal space between chairs, none at ends
  • space-around: Equal space around each chair
  • space-evenly: Equal space between and around chairs

üí° Key Insight: justify-content only affects spacing along the main axis. If your flex-direction is row, it controls horizontal spacing. If it's column, it controls vertical spacing!

/* The container's main axis (red line above) */
.container {
  display: flex;
  flex-direction: row; /* ‚Üê This sets the main axis */
  justify-content: flex-start; /* ‚Üê This distributes along it */
}

Remember: If you change flex-direction: column, the main axis becomes vertical, and justify-content will distribute items vertically!

Practice Challenge: Perfect Navigation

Your Mission: Create the perfect navigation bar layout that web developers use daily.

  1. Start with a container containing a logo and 3 navigation items
  2. Use justify-content to position the logo on the left and nav items on the right
  3. Make all items vertically centered
  4. Add appropriate spacing between navigation items
  5. Test different justify-content values to see which one works best

Hint: You'll need to use space-between or combine multiple techniques!

LOGO
Home
Products
Contact
/* Solution 1: Using space-between */
.navbar {
  display: flex;
  justify-content: space-between; /* Logo left, items right */
  align-items: center; /* Vertically center */
}

.nav-items {
  display: flex;
  gap: 20px; /* Space between nav items */
}

/* Or Solution 2: Using auto margins */
.navbar {
  display: flex;
  align-items: center;
}

.logo {
  margin-right: auto; /* Pushes everything else to the right */
}

.nav-items {
  display: flex;
  gap: 20px;
}
