Introducing Multi-line Flexbox

flex-wrap and gap are CSS properties that transform single-line flex containers into responsive, multi-line layouts. While basic flexbox handles one row or column, these properties allow items to flow naturally across multiple lines like text wrapping.

ðŸ“š Bookshelf Analogy

Imagine arranging books on a shelf:

  • nowrap (default): Try to fit all books on one shelf, even if they hang off the edges
  • wrap: Books that don't fit move to the next shelf automatically
  • wrap-reverse: Books fill from bottom shelf up instead of top down
  • gap: Consistent spacing between books, making them easier to browse

ðŸ’¡ The Responsive Breakthrough: Before gap, developers used margins to space flex items, which created awkward edge cases. The gap property gives us perfect, consistent spacing that only appears between items, not on the edges. Combined with flex-wrap, this creates truly responsive layouts!

Practice Challenge: Responsive Photo Gallery

Your Mission: Create a responsive photo gallery that works on any screen size.

  1. Start with a container holding multiple photo items
  2. Make photos wrap to new lines when they run out of space
  3. Add consistent spacing between photos (both horizontally and vertically)
  4. Make each photo flexible with a minimum width of 150px
  5. Test by adjusting the container width in the demo above
  6. Extra credit: Make photos maintain a square aspect ratio

Hint: You'll need flex-wrap: wrap, gap, and flex: 1 1 150px on the items!
