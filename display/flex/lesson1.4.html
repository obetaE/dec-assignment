Today's Question

How do we align items along the cross axis (vertical alignment)?

The missing piece: align-items

Introducing align-items

align-items is a CSS property that controls how flex items are aligned along the cross axis of the flex container. While justify-content handles main axis distribution, align-items handles cross axis alignment - the "vertical" alignment for horizontal layouts!

üè¢ Office Building Analogy

Imagine arranging people in an office with rows of desks:

  • stretch (default): Everyone stands tall, touching floor and ceiling
  • flex-start: Everyone stands at the top of their space
  • flex-end: Everyone stands at the bottom of their space
  • center: Everyone stands in the middle of their space
  • baseline: Everyone aligns their name tags at the same height

üí° The Magic of Center: For years, vertically centering elements in CSS was notoriously difficult. With align-items: center in a flex container, it becomes trivial! This one property solves a major historical CSS pain point.

The Magic of Perfect Centering

Historical CSS Problem Solved!

For years, vertically centering elements was one of the hardest things in CSS. Flexbox makes it trivial with just two lines:

Before Flexbox (Hard!)

Multiple techniques, all complicated:

Centered
/* Multiple techniques needed */
.centered {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

Complex, breaks flow, positioning issues

With Flexbox (Easy!)

Two simple properties:

Centered
/* Simple flexbox centering */
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Simple, maintains flow, responsive

The Flexbox Centering Formula: display: flex; + justify-content: center; + align-items: center; = Perfect centering in both directions!

Special Value: baseline

Text Alignment

baseline is a special alignment value that aligns items based on their text baseline (the invisible line text sits on).

When to use it:

  • Forms with labels and inputs
  • Navigation with different font sizes
  • Anywhere text needs to line up regardless of element size
/* Align form labels and inputs by text */
.form-group {
  display: flex;
  align-items: baseline;
  gap: 10px;
}

Practice Challenge: Perfect Card Grid

Your Mission: Create a card grid where cards have equal height regardless of content length.

  1. Start with three cards containing different amounts of content
  2. Use align-items to make all cards the same height
  3. Center the card content vertically within each card
  4. Make sure the cards are spaced evenly horizontally
  5. Test different align-items values to see which one gives you equal heights

Hint: You'll need to use align-items: stretch on the container and then center content within each card!

Card 1
Short content
Card 2
Medium length content that might wrap to multiple lines if needed
Card 3
Very short
/* Solution for equal height cards */
.card-grid {
  display: flex;
  align-items: stretch; /* Makes all cards same height */
  justify-content: space-between; /* Even horizontal spacing */
  gap: 20px;
}

.card {
  flex: 1; /* Cards take equal width */
  display: flex; /* Nested flex for card content */
  flex-direction: column;
  justify-content: center; /* Center content vertically within card */
  align-items: center; /* Center content horizontally within card */
  padding: 20px;
  background: white;
  border-radius: 10px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}
