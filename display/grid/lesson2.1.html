Welcome to Two-Dimensional Layouts

CSS Grid is a layout system that allows you to create complex two-dimensional designs with both rows and columns. Unlike flexbox (which handles one axis at a time), grid lets you define the entire structure upfront.

The magic happens when you declare display: grid on a container. This single property transforms all direct children into grid items that can be placed anywhere within the grid you define.

üèóÔ∏è Construction Blueprint Analogy

Imagine building a house:

  • Flexbox: Like arranging furniture along a single wall (one dimension)
  • Grid: Like drawing the entire house blueprint with rooms laid out in rows and columns (two dimensions)
  • grid-template-columns: Defines how many rooms per row and their widths
  • grid-template-rows: Defines how many rooms per column and their heights

üí° The Grid Mindset: With CSS Grid, you define the structure first (rows and columns), then place items within that structure. This is opposite to flexbox where items dictate the layout. Grid gives you precision control over the entire layout from the start.

The Grid Declaration Syntax

Creating Grids

Creating a CSS Grid starts with three essential properties:

1. Activate Grid

The foundation property that enables grid layout

.container { display: grid; /* or inline-grid */ }

This alone creates a 1x1 grid (one column, one row)

2. Define Columns

Set up your column structure

.container { grid-template-columns: 200px 200px 200px; /* Three columns, each 200px wide */ }

Each value = one column width

3. Define Rows

Set up your row structure (optional)

.container { grid-template-rows: 100px 100px; /* Two rows, each 100px tall */ }

If omitted, rows are created automatically

Complete Example

A 3√ó2 grid with fixed dimensions

.container { display: grid; grid-template-columns: 150px 150px 150px; grid-template-rows: 80px 80px; }

Creates a perfect 3-column, 2-row grid

Understanding Grid Lines

The Coordinate System

CSS Grid uses an invisible coordinate system of grid lines to position items. Each line is numbered starting from 1:

Column Lines

Vertical lines that separate columns. A 3-column grid has 4 column lines (1 before, 1 between each column, 1 after).

/* 3 columns = 4 column lines */ grid-template-columns: 100px 100px 100px; // Line 1 Line 2 Line 3 Line 4

Row Lines

Horizontal lines that separate rows. A 2-row grid has 3 row lines (1 before, 1 between rows, 1 after).

/* 2 rows = 3 row lines */ grid-template-rows: 100px 100px; // Line 1 Line 2 Line 3

Grid Lines are Your Coordinates: You'll use these line numbers to place items anywhere in the grid (e.g., "start at column line 2, end at column line 4").
