Introducing the Fractional Unit

The fr unit (fractional unit) is CSS Grid's secret weapon for creating flexible, responsive layouts. It represents a fraction of the available space in the grid container.

Unlike pixels (fixed) or percentages (relative to parent), fr units distribute only the free space after accounting for fixed-sized tracks and gaps. This makes them perfect for responsive design!

üçï Pizza Analogy

Imagine you have a pizza (the grid container) to share with friends:

  • Pixels (px): Each friend gets exactly 2 inches of pizza, regardless of pizza size
  • Percentages (%): Each friend gets 25% of the pizza, but if you cut off the crust first, the math gets complicated
  • fr units: You divide the entire pizza equally among friends. Add more friends? Each slice automatically gets smaller!

üí° The fr Unit Magic: The browser calculates fr values after allocating space for fixed-size items and gaps. This means you can mix fixed and flexible tracks in the same grid, and everything just works!

CSS Units Comparison

Which Unit When?

Different units serve different purposes in CSS Grid. Here's when to use each:

px (Pixels)

Fixed size - doesn't change

200px 200px

Perfect for fixed elements like logos, icons

% (Percentages)

Relative to parent - includes gaps in calculation

33% 33% 33%

Tricky with gaps: 33% + 33% + 33% + gaps > 100%

fr (Fractional)

Fraction of free space - ignores gaps in calculation

1fr 1fr 1fr

Perfect for responsive layouts - automatically adjusts

auto

Content-based - sizes to fit content

auto 1fr auto

Great for sidebar + main content layouts

The fr Advantage: With fr units, the browser handles all the complex math. You just define the proportions, and the grid automatically fills the available space perfectly!
