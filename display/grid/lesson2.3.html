Perfect Spacing and Positioning

Spacing and alignment transform basic grids into polished, professional layouts. While grid structure defines where items go, spacing controls the breathing room between them, and alignment determines their exact position within each cell.

CSS Grid offers two types of alignment: item alignment (within cells) and content alignment (of the entire grid within its container). Plus, the gap property gives you perfect, consistent spacing that eliminates the need for messy margin hacks.

üñºÔ∏è Picture Gallery Analogy

Imagine hanging pictures in a gallery:

  • gap: Consistent spacing between picture frames (gutters)
  • align-items: How each picture sits within its frame (top, center, bottom)
  • justify-items: Horizontal positioning of pictures within frames (left, center, right)
  • align-content & justify-content: How the entire gallery wall is positioned in the room

üí° The Grid Gap Advantage: Unlike margins, gap only creates space between items, never on the outer edges. This eliminates the need for negative margins or special selectors like :first-child and :last-child!

Understanding Justify vs Align

Two Axes

CSS Grid has two alignment axes. Understanding them is key to mastering alignment:

Main Axis (Justify)

Controls alignment along the inline (horizontal) direction by default

Main Axis
justify-*
.container { justify-items: center; justify-content: center; }

Horizontal alignment (left/right/center)

Cross Axis (Align)

Controls alignment along the block (vertical) direction by default

Cross Axis
align-*
.container { align-items: center; align-content: center; }

Vertical alignment (top/bottom/center)

Simple Memory Aid: Think of justify as "justify text" (left/right/center = horizontal). Think of align as "align vertically" (top/center/bottom = vertical).

Practice Challenge: Perfect Photo Gallery

Your Mission: Create a perfectly aligned photo gallery with consistent spacing.

  1. Create a 3-column grid with 6 photos
  2. Add 20px spacing between all photos (both rows and columns)
  3. Center all photos within their grid cells
  4. Make sure the entire gallery is centered in its container
  5. Give photos different heights to test alignment
  6. Extra credit: Use different row and column gaps (15px vertical, 25px horizontal)

Hint: You'll need gap or row-gap and column-gap, place-items: center, and place-content: center!

  /* Center photos within their cells */
